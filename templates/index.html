Welcome to MS Learn Formatter

This tool reformats Microsoft Learning modules for better readability while preserving navigation.

How to use:

  1. Enter a Microsoft Learning module URL above
  2. Click "Load Module" to fetch and format content
  3. Navigate through sections using the sidebar

Features:

  • Clean, distraction-free formatting
  • Preserved module structure and navigation
  • Mobile-responsive design
  • Real-time content loading with HTMX
  • Local Flask server bypasses CORS restrictions

Example URLs to try:

  • https://learn.microsoft.com/en-us/azure/
  • https://docs.microsoft.com/en-us/learn/modules/
  • Any Microsoft Learn module URL
